# The Collaboration Thesis

**Why delegating everything to AI agents is a trap — and what to build instead.**

Everyone is building the same thing: assign a task, dispatch an agent, get a pull request. But delegation makes humans dumber over time. When models commoditize, the only differentiator is how well humans collaborate with agents.

```
Result = Human judgment × Agent capability

Agent capability → commodity (same for everyone)
Human judgment → the only variable that matters
```

Read the full thesis: **[MANIFESTO.md](./MANIFESTO.md)**

## The Bet

We're betting that the future isn't "agents work for me" — it's "agents work with me."

The infrastructure for this doesn't exist yet. Current tools handle task dispatch. Nothing handles:

- Decision transparency (why did the agent choose this approach?)
- Structured handoffs (agent needs human judgment — here's the context)
- Shared context (persistent knowledge both human and agent build on)
- Progress contracts (agent proposes plan, human approves, checkpoints along the way)
- Team identity (who is this agent, what can it do, what's its track record?)
